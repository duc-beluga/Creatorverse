# WEB103 Prework - _👉🏿 CREATORVERSE_

Submitted by: **👉🏿 Duc Nguyen**

About this web app: **👉🏿 This is a full-stack web app designed to showcase information about well-known people. It allows users to add, view, edit, or delete profiles. Each profile includes the person's name, a link to their Wikipedia page, an image URL, and a brief description about them.**

The core technologies used to build this app include:

- **React** for the frontend user interface
- **Tailwind** for styling
- **FontAwesome** for icons
- **Supabase** for backend services and database management.

Time spent: **👉🏿 5** hours
